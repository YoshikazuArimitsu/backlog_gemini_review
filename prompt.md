# 動作要件

- Windows環境で標準のPython・Powershellで動作します。
- Backlog・Gitの操作はAPIキーを設定ファイルから読み込み、Backlog APIを利用して行ってください。
- Gemini の操作は Chrome のログイン済アカウントを使用し、ブラウザ操作で行ってください。

# 機能要件

## 1. PR/Gitの差分取得

- Backlog の指定されたプロジェクト・PR番号からPRコメント・ベースとの差分を取得してください
- 取得した情報を元に Gemini にレビューを依頼するプロンプトを作成してください
- 作成したプロンプト文はクリップボードにもコピーし、手動で利用できるようにしておいてください

## 2. Geminiによるレビュー

- Chrome を起動し、Gemini の Chat を開き、Proモードに切り替えてください
- 前の段階で取得したプロンプト文を渡し、回答を取得してください。
- 取得した回答を画面に表示し、MarkDown テキストでファイルに保存してください

## 3. 一括実行スクリプト

- 上記の手順を一括で実行するスクリプトを作成してください。
